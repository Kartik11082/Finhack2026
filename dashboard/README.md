# AI Hype Decoded: The Prediction Project
**Does the structure of executive AI-speak meaningfully predict future equity return anomalies?**

This dashboard visualizes the findings of an end-to-end Machine Learning pipeline that isolates and measures the pure predictive alpha generated by Natural Language Processing (NLP) against traditional stock momentum factors.

## High-Level Data Analysis Methodology

We constructed an A/B testing framework using two discrete Random Forest mathematical models. By holding the algorithm, hyperparameters, and cross-validation schema identical, any divergence in accuracy is mathematically isolated directly to the introduction of the linguistic sentiment vectors.

### 1. The Baseline Strategy (Price-Only)
We first established a "dumb" baseline model using only easily accessible, conventional quantitative factors. For 9 Mega-Cap Tech equities over approximately ~200 historical earnings events (2020-2025), we extracted:
- **7-Day Price Momentum**
- **7-Day Volatility**
- **Volume Spikes**
- **Underlying Market Return (SPY baseline)**

### 2. The Enhanced Strategy (NLP Injection)
To test the "Hype" thesis, we built two independent text-extraction pipelines and ran the raw string structures through **ProsusAI/FinBERT** (a BERT model explicitly fine-tuned for financial sentiment analysis).

*   **Executive Transcript Pipeline**: Using historical earnings call datasets (via Kaggle/SEC), we mapped exact calendar quarter text files to their earnings dates. Crucially, the parser isolates the trailing ~40% of the transcript—specifically targeting the unscripted **Analyst Q&A** sections where executives are most likely to leak genuine forward-looking sentiment.
*   **Media Buzz Pipeline**: We extracted historical headline snapshots from Finviz to measure retail and institutional "Buzz". 

The text is divided into 512-token chunks, scored algorithmically for Bullish/Bearish drift, and averaged to form unified `transcript_sentiment` and `buzz_score` vectors.

### 3. Time-Aware Cross-Validation
Predicting stocks is notoriously susceptible to "Data Leakage" (training a model on data from 2024 to predict a stock move in 2022). 

To enforce absolute academic rigor, both models employ `TimeSeriesSplit`. The ~214 events are chronologically sorted and split into 5 Folds. The model *never* sees future data during its training phase; it strictly rolls forward through time, updating its predictive weights, and evaluating itself purely on the unseen future events immediately ahead of its timeframe.

### 4. Evaluating the Alpha
The results are pushed mathematically into standard `JSON` payloads consumed by this React presentation deck. We evaluate the true edge of the "AI Hype" by comparing:
- **Baseline Accuracy vs Enhanced Accuracy**
- **ROC-AUC & F1 Scoring** (proving the model isn't just universally guessing "UP" on tech stocks in a bull market).
- **Feature Importance**: Natively querying the Random Forest matrix to verify if the NLP transcript factor successfully forced its way past Price Momentum to the top of the decision tree.
